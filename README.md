## üìñ Usage Guide

### 1. Adding Music & Setup

- Go to **Settings** (Gear icon).
- Under "Library Paths", click **"Add Folder"**.
- Select the folder on your computer where your music is stored.
- Click **"Scan Library"** to import your tracks.

### 2. Using the Equalizer

- Navigate to the **Equalizer** tab from the sidebar.
- Toggle the switch to **"Active"**.
- Adjust the sliders to your preference or select a **Preset** from the dropdown.

### 3. Using Radio

- Go to the **Radio** tab.
- Use the **Search bar** to find specific stations.
- Use the **Globe icon** to filter by Country.
- Click any station card to start streaming instantly.

### 4. Mini Mode

- Click the **Minimize/Expand** icon in the bottom playback bar (or top window controls).
- The player will shrink to a compact widget.
- Click **"EXPAND"** on the mini player to return to the full application.
